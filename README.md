Overview
This project focuses on Sentiment Analysis, a Natural Language Processing (NLP) task that determines the emotional tone behind text data. By analyzing textual input, the system categorizes the sentiment as positive, negative, or neutral. This can be applied to various domains such as customer feedback analysis, social media monitoring, and market research.

Features
Preprocessing: Cleans and prepares text data (e.g., tokenization, stopword removal, stemming/lemmatization).
Sentiment Classification: Predicts the sentiment of the given text using random forest algorithm
Visualization: Displays results through charts and graphs for better insights.
